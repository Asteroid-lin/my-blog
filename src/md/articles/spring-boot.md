# Spring Boot 核心技术

## 概述

Spring Boot 是一个基于 Spring 框架的快速开发脚手架，它简化了 Spring 应用的配置和部署过程。

## 核心特性

### 1. 自动配置
- 基于类路径和 Bean 定义自动配置
- 减少 XML 配置
- 约定优于配置

### 2. 起步依赖
- 预定义的依赖组合
- 版本兼容性管理
- 快速项目搭建

### 3. 内嵌服务器
- 内置 Tomcat、Jetty 等服务器
- 独立运行，无需外部容器
- 简化部署流程

## 最佳实践

1. **配置管理**
   - 使用 `application.yml` 进行配置
   - 环境特定配置分离
   - 配置属性验证

2. **异常处理**
   - 全局异常处理器
   - 自定义异常类型
   - 统一错误响应格式

3. **监控和健康检查**
   - Actuator 端点
   - 自定义健康检查
   - 指标收集和监控
